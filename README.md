# Proyecto: Análisis de Factores de Ventas con IA

## 1. Objetivo y Pregunta Central
El objetivo de este proyecto es utilizar técnicas de minería de datos para responder a la pregunta: **¿Qué factores influyen más en las ventas de un producto?** Se busca entender cómo la inversión en publicidad y los descuentos afectan el volumen de unidades vendidas.

## 2. Fuente de Datos y Limpieza
- **Fuente:** Dataset sintético generado con Python para simular un escenario retail.
- **Tamaño:** 500 registros.
- **Limpieza:** Se verificó la ausencia de valores nulos y se estandarizaron los formatos numéricos. El dataset final se encuentra en este repositorio como `dataset_ventas_limpio.csv`.

## 3. Técnicas de Minería de Datos
Se utilizó **Python (Scikit-Learn)** para realizar un análisis de **Regresión Lineal**.
- **Variables predictoras:** Inversión en Publicidad, Porcentaje de Descuento.
- **Variable objetivo:** Unidades Vendidas.

## 4. Resultados y Conclusiones
Gracias al apoyo de la IA en el análisis de coeficientes, se determinó que:
1. Existe una **correlación positiva fuerte** entre la publicidad y las ventas.
2. Por cada unidad monetaria invertida en publicidad, las ventas aumentan aproximadamente en 0.5 unidades.
3. Los descuentos tienen un impacto significativo, aumentando las ventas en aprox. 20 unidades por cada punto porcentual de descuento.

## 5. Visualización
Se generaron gráficos de dispersión (Scatter plots) que confirman visualmente la tendencia lineal ascendente entre la inversión y el volumen de ventas.
